01 Introduction

This Privacy Policy describes how ScheduleX ("the Application") handles information in connection with its operation. ScheduleX is a personal productivity tool built for a single user to schedule and publish videos to their own TikTok account via TikTok's official API.

Because ScheduleX is a strictly personal, single-user tool, this policy is simple: no personal data is collected, stored, or shared with any third party.

02 Data Accessed

To function, ScheduleX accesses the following data from TikTok's API — all of which belongs exclusively to the sole user of the application:

Data Type Purpose Stored? Shared?
TikTok OAuth Token Authenticate API requests to publish videos Session only Never
Display name & Avatar Show connected account in the UI Session only Never
Video list Avoid publishing duplicate content Not stored Never
Uploaded video files Temporary storage before scheduled publication Locally, temp Never
Publication job data Track scheduled posts in jobs.json Local only Never

03 How Data Is Used

All data accessed by ScheduleX is used exclusively for the following purposes:

  • Authenticating API calls to TikTok on behalf of the user
  • Displaying the connected account's name and avatar in the interface
  • Temporarily storing scheduled video files until they are published and subsequently deleted
  • Maintaining a local queue of scheduled publication jobs

No data is used for advertising, analytics, profiling, machine learning, or any purpose other than the core functionality described above.

04 Data Storage & Retention

ScheduleX stores data exclusively on the user's own machine (localhost):

  • OAuth tokens are stored in the server-side session and expire automatically (access token: 24h, refresh token: 365 days)
  • Video files are deleted automatically after successful publication or manual cancellation
  • Job records are stored in a local jobs.json file on the user's own machine and can be deleted at any time

ScheduleX does not use any external database, cloud storage, or remote server. All data remains on the user's local machine at all times.

05 Data Sharing

ScheduleX does not share any data with any third party, with the following necessary exception:

  • TikTok Inc. — Video files and metadata are transmitted to TikTok's servers exclusively for the purpose of publication, via TikTok's official Content Posting API. This is the intended and sole purpose of the Application.

No data is shared with advertisers, analytics providers, data brokers, or any other entity.

06 Cookies & Tracking

ScheduleX does not use cookies, tracking pixels, analytics scripts, or any form of user behavior monitoring. A server-side session is used solely to maintain the OAuth authentication state during a browsing session.

07 Third-Party Services

ScheduleX interacts exclusively with TikTok's official API (open.tiktokapis.com). No other third-party service, SDK, or external API is used. The privacy practices of TikTok Inc. are governed by TikTok's own Privacy Policy, available at tiktok.com/legal/privacy-policy.

08 Children's Privacy

ScheduleX is not intended for use by individuals under the age of 13. As a personal tool, it does not collect any data from any user, including minors.
